### 📊 Dashboard de Controle de Perdas Operacionais

Projeto desenvolvido para analisar perdas de bebidas e apoiar o controle de estoque. A análise permitiu identificar produtos críticos, padrões de desperdício e oportunidades de melhoria, contribuindo para uma redução aproximada de **40% nos prejuízos operacionais relacionados a bebidas**.

#### Problema de negócio

O controle anterior dificultava a identificação rápida dos produtos que mais geravam perdas. O objetivo foi transformar os registros operacionais em indicadores claros para apoiar ações de redução de desperdícios.

#### Principais análises

- Valor e quantidade das perdas
- Produtos com maior impacto financeiro
- Participação percentual de cada item nas perdas
- Monitoramento de estoque e consumo
- Identificação de gargalos operacionais

#### Tecnologias e recursos

- Power BI
- Power Query
- DAX
- Modelagem Star Schema
- KPIs operacionais
- Dashboard executivo

#### Resultado

O projeto demonstrou como a análise estruturada dos dados pode orientar ações de controle, priorização e melhoria operacional.
